# Pension Tax Relief Calculator - Usage Guide

## Getting Started

### Step 1: Check Basic Relief
1. Open `basic_rate_relief.csv`
2. Enter your contribution
3. See automatic 20% added

### Step 2: Claim Higher Rate Relief
1. Open `higher_rate_relief.csv`
2. Check if you're higher/additional rate
3. Calculate extra relief to claim

### Step 3: Check Annual Allowance
1. Use `annual_allowance.csv` to track your £60k limit
2. If over £260k income, check `tapered_allowance.csv`

### Step 4: Plan Retirement
1. Use `retirement_projection.csv` to estimate pot size
2. Use `carry_forward.csv` if you have unused allowance
