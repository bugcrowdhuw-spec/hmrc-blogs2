## Key Pension Facts

- **Basic Rate:** 20% automatic (added to pot)
- **Higher Rate:** 20% extra claimable via tax return
- **Additional Rate:** 25% extra claimable
- **Annual Allowance:** £60,000 (2025/26)
- **Tapered:** Reduces for £260k+ income
