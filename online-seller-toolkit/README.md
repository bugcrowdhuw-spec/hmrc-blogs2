## Quick Start

1. Open `income_tracker.csv` in Excel or Google Sheets
2. Add your platforms (Vinted, eBay, Etsy, etc.)
3. Record each sale - note if personal clearout or sourcing
4. Use `expenses_calculator.csv` for costs
5. Check `trading_allowance.csv` to see if you owe tax
6. Review `tax_year_checklist.md` before 5 April

## Key Tax Facts

- **Trading Allowance:** First £1,000 tax-free
- **VAT Threshold:** Register when over £90,000/year
- **CGT:** Pay on profit from items sold for more than bought
- **Platform Reporting:** HMRC receives data from eBay, Etsy, Vinted
