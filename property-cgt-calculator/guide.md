# Property CGT Calculator - Usage Guide

## Getting Started

### Step 1: Identify Property Type
- Main residence → Use `main_residence.csv`
- Buy-to-let → Use `buy_to_let.csv`
- Second home → Use `additional_property.csv`
- Inherited → Use `inherited_property.csv`

### Step 2: Calculate Your Gain
1. Enter purchase price and costs
2. Enter sale price and costs
3. Add any capital improvements
4. Calculate gross gain
5. Apply any reliefs
6. Deduct annual exemption

### Step 3: Report and Pay
1. Use `reporter_60day.csv` to track deadline
2. Report to HMRC within 60 days
3. Pay any CGT due

---

## Key CGT Rules

| Rule | Detail |
|------|--------|
| Annual Exemption | £3,000 |
| Basic Rate | 18% |
| Higher Rate | 24% |
| 60-Day Rule | Report within 60 days |
| Additional Property | +3% surcharge |

---

## Common Mistakes

1. Forgetting to report within 60 days
2. Not using annual exemption
3. Missing improvements as deduction
4. Wrong property type selected
5. Not calculating final period relief
