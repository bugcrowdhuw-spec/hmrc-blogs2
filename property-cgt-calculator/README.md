## Quick Start

1. Identify property type (main residence, buy-to-let, second home)
2. Open relevant calculator file
3. Enter purchase price and sale price
4. Add improvements and costs
5. Calculate CGT liability
6. Use `reporter_60day.csv` to meet deadline

---

## Key CGT Facts

- **Annual Exemption:** £3,000 (2025/26)
- **CGT Rates:** 18% basic, 24% higher (Oct 2024+)
- **Main Residence Relief:** Final 9 months exempt
- **Additional Property Surcharge:** 3%
- **60-Day Reporting:** Must report within 60 days of sale
