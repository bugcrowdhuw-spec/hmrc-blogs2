## Quick Start

1. Open `dividend_tracker.csv` in Excel or Google Sheets
2. Add your investments (ISA, General, Pension)
3. Record dividend payments
4. Use `interest_tracker.csv` for savings
5. Track CGT in `cgt_calculator.csv`
6. Check `annual_summary.csv` for tax return

## Key Tax Facts

- **Dividend Allowance:** First £500 tax-free
- **Dividend Tax Rate:** 8.75% (basic), 33.75% (higher), 39.35% (additional)
- **CGT Annual Exemption:** £3,000 (2025/26)
- **CGT Rates:** 10% (basic), 20% (higher/additional)
- **Crypto:** Now subject to CGT
