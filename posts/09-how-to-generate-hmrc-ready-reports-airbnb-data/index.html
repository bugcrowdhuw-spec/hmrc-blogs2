How to Generate HMRC-Ready Reports from Airbnb Data

Your Airbnb earnings must be reported to HMRC — but what format do they need? Here’s how to transform your hosting data into reports that pass compliance checks.

Understanding HMRC Report Requirements

HMRC expects property income in specific formats:

Self Assessment Requirements

Your annual tax return requires:

  • Total income — All earnings from property rental
  • Total expenses — Allowable deductions
  • Net profit — Income minus expenses
  • Supporting details — Property-by-property breakdown

MTD Requirements

If you’re VAT-registered or voluntarily in MTD:

  • Quarterly submissions — Every 3 months
  • Digital records — All transactions captured
  • API filing — Direct to HMRC systems

XML Report Format

HMRC prefers machine-readable reports:

  • Structured data files — XML format preferred
  • Field-by-field breakdown — Detailed transaction data
  • Validation requirements — Must pass HMRC checks
  • Upload capability — Direct to government gateway

Airbnb Data Sources

Where to Find Your Data

Airbnb provides several reporting documents:

  • Host Dashboard — earnings summary, booking history
  • Transaction History — detailed payout records
  • Year-end Tax Document — annual earnings summary
  • Payout Statements — bank transfer records

Extracting Your Data

  1. Log into Airbnb — Access host account
  2. Go to Account — Settings and personal info
  3. Select Taxes — Tax documents and reports
  4. Download reports — Choose date ranges
  5. Save files — Store for importing

Creating HMRC-Ready Reports

Step 1: Compile Income Data

Start with total earnings:

  • Pull year-end summary — From Airbnb tax documents
  • List all payouts — Bank statement verification
  • Add direct payments — Guests paying on arrival
  • Include other income — Related property earnings

Step 2: Categorise Expenses

Airbnb-specific expenses to capture:

  • Host fees — Airbnb platform charges
  • Cleaning costs — Between guests and weekly
  • Linen and toiletries — Guest supplies
  • Utilities — Higher usage from rentals
  • Insurance — Host insurance policies
  • Repairs and maintenance — Keeping property in good order
  • Mortgage interest — If applicable
  • Agent fees — Property management costs

Step 3: Generate Reports

Using dedicated software:

  1. Import platform data — Automated connection
  2. Categorise transactions — Automatic coding
  3. Review and adjust — Ensure accuracy
  4. Generate XML — One-click report creation
  5. Export for filing — Direct to HMRC

Using Spreadsheets

If using spreadsheets (not recommended):

  1. Create income tab — List all earnings
  2. Create expense tab — Categorise deductions
  3. Add formulas — Calculate net profit
  4. Map to HMRC fields — Structure appropriately
  5. Export to CSV — Convert for upload

Common Report Errors

Avoid these mistakes when generating reports:

  • Missing platform fees — Always deductable
  • Wrong expense categories — HMRC rejects improper coding
  • Incorrect dates — Tax year vs calendar year
  • Duplicate entries — Same income counted twice
  • Inconsistent totals — Platform vs declared doesn’t match
  • Late submission — Missing quarterly or annual deadlines

FAQ

Does Airbnb send data directly to HMRC?

Since 2023, digital platforms report earning data to HMRC under OECD rules. This information is shared automatically. Your Self Assessment should match platform reports.

Can I generate reports without special software?

Technically yes, but spreadsheets carry significant risks: higher error rates, no validation, no audit trail. HMRC-approved software is strongly recommended.

What if my reports don’t match platform data?

Investigate immediately. Find the discrepancy and correct your records. Platform data and Self Assessment don’t match triggers HMRC flags.

How do I submit XML reports to HMRC?

Use HMRC-approved software to generate XML directly. File through your software’s MTD integration or upload via your Government Gateway account.
