What Property Managers Need to Know About MTD

Property managers face unique MTD challenges. Here’s what you need to know to stay compliant while managing multiple rental properties.

The Property Manager’s MTD Challenge

Property managers handle complexity that individual landlords don’t:

  • Multiple properties — Different owners, different structures
  • Multiple platforms — Airbnb, Booking.com, Vrbo, direct
  • Multiple income streams — Rents, fees, deposits
  • Multiple expense types — Repairs, management, legal
  • Multiple tax rules — Residential, holiday let, HMO

MTD software must handle all this complexity.

MTD Applicability

Who Must Comply

Property managers must comply if:

  • Your company is VAT registered — Exceeds £85,000
  • Your clients are VAT registered — You handle their tax
  • Voluntary MTD — Chose to join early

Even if not mandatory: MTD benefits property managers significantly. Real-time visibility helps client management.

Client Responsibilities

If managing for clients:

  • Clarify responsibilities — Who’s responsible for MTD?
  • Agree on software — Use compatible systems
  • Establish workflows — How data flows between you
  • Document agreements — Written procedures

Digital Record Requirements

Property-Level Tracking

Each property needs:

  • Income tracking — By property, by period
  • Expense allocation — Direct vs apportioned
  • Profit calculation — Per property
  • Supporting documents — Receipts, agreements

Client Data Separation

For property management:

  • Access controls — Who sees what
  • Audit trails — Track all access

What Property Managers Submit

Each quarter:

  • Total rental income received — For all properties
  • Total allowable expenses — Property costs
  • Tax estimated — Based on profit
  • VAT position — If applicable

Review Before Filing

Before quarterly submission:

  1. Reconcile all transactions — Verify completeness
  2. Check expense categorisation — Ensure proper coding
  3. Verify property allocation — Correct property
  4. Review client statements — Confirm accuracy
  5. File on time — Meet deadline

Choosing Software

Consider:

  • Portfolio size — Match to software limits
  • Client count — Number managed
  • Platform mix — All your channels
  • MTD approval — Must be on HMRC list
  • Cost — Subscription vs per-property fees

Implementation Guide

Step 1: Audit Your Position

  1. List all properties — Know what you’re managing
  2. Document income sources — All platforms
  3. Identify expense types — Categories you use
  4. Assess current records — Paper vs digital

Step 2: Choose Software

  1. Compare property management tools — Look for MTD approval
  2. Check integrations — Platform connections
  3. Test with sample data — Verify functionality
  4. Assess costs — Ongoing fees

Step 3: Set Up System

  1. Configure properties — Add all rentals
  2. Connect platforms — Authorise integrations
  3. Set up expense categories — Match your needs
  4. Configure client access — Permissions

Step 4: Train and Migrate

  1. Train team — Ensure adoption
  2. Migrate historical data — Import past records
  3. Test quarterly process — Run test returns
  4. Refine workflows — Optimize

FAQ

Do property management companies need their own MTD compliance?

Yes, if the company is VAT registered or voluntarily in MTD. Each business entity must comply separately.

Can clients file their own MTD while I manage properties?

Yes. Many clients handle their own tax. Your role is providing accurate reports. Clarify responsibilities in your management agreement.

What happens if property management is new to MTD?

Start now. Even if not mandatory, early adoption builds good habits and provides real benefits. HMRC’s pilot programme is recruiting volunteers.

How do I handle clients who haven’t complied?

If clients aren’t MTD-ready, help them select compliant software. Your management reports must still work with their filing requirements.
