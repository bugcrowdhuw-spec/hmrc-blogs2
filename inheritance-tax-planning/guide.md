# Inheritance Tax Planning - Usage Guide

## Getting Started

### Step 1: Calculate Potential IHT
1. Open `iht_estimator.csv`
2. Enter all assets and liabilities
3. See estimated IHT liability

### Step 2: Check Nil-Rate Band
1. Use `nil_rate_band.csv` to track unused allowance
2. If married, can transfer partner's unused band

### Step 3: Plan Gifts
1. Use `gift_calculator.csv` for 7-year rule
2. Track gifts and years since
3. Plan gifting strategy

### Step 4: Check Residence Allowance
1. Use `residence_allowance.csv` for main home
2. Check if over £2m estate (tapered)

---

## Key IHT Rules

| Threshold | Amount |
|-----------|--------|
| Nil-Rate Band | £325,000 |
| Residence Allowance | £175,000 |
| Combined (married) | £650,000 + £350,000 |
| IHT Rate | 40% |

---

## Planning Strategies

1. **Gift early** - Use 7-year rule
2. **Use annual exemptions** - £3,000/year
3. **Business relief** - 100% on trading assets
4. **Pensions** - Usually IHT-free
5. **Trusts** - For complex estates

---

## Common Mistakes

1. Not using spouse's nil-rate band
2. Missing 7-year rule on gifts
3. Not claiming residence allowance
4. Forgetting business relief
5. Gifting too close to death
