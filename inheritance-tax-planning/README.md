## Key IHT Facts

- **Nil-Rate Band:** £325,000 (frozen)
- **Residence Allowance:** £175,000 (frozen, tapering)
- **IHT Rate:** 40% above threshold
- **7-Year Rule:** Gifts fully exempt after 7 years
- **Annual Gift Allowance:** £3,000/year
