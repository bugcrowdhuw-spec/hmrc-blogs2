# Crypto Tax Calculator

**For UK Crypto Investors**  
**Tax Year 2025/26**

---

## What's Included

| File | Description |
|------|-------------|
| `cost_basis.csv` | Track purchase price and cost basis |
| `cgt_calculator.csv` | Calculate capital gains |
| `income_tracker.csv` | Mining, staking, airdrops |
| `exchange_log.csv` | Track all transactions |
| `guide.md` | Complete usage guide |

---

## Quick Start

1. Track purchases in `cost_basis.csv`
2. Record sales in `cgt_calculator.csv`
3. Log mining/staking in `income_tracker.csv`

---

## Key Crypto Tax Facts

- **CGT Rate:** 10% basic, 20% higher
- **Annual Exemption:** £3,000
- **Income:** Mining/staking taxed as income
