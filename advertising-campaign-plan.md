# HMRC Reporter — Advertising Campaign Plan

**Prepared:** April 2026  
**Product:** HMRC Reporter (hmrcreporter.com)  
**Context:** SaaS automating tax compliance for UK short-term rental hosts

---

## 1. Target Audience

### Primary Segments

| Segment | Profile | Pain Points | Priority |
|---------|---------|-------------|----------|
| **Airbnb/Short-Term Rental Hosts** | Individual property owners with 1-5 properties | Spreadsheet fatigue, MTD confusion, multi-platform data reconciliation | HIGH |
| **Holiday Let Landlords** | Professional operators with 5+ properties | Volume processing, compliance at scale, quarterly reporting | HIGH |
| **Property Managers** | Companies managing 10+ properties for owners | Client reporting, data aggregation, bulk submissions | HIGH |
| **Accountants** | Practices serving property/Holiday Let clients | Manual data processing, client time sink, scalability | HIGH |

### Secondary Segments

- **New Airbnb hosts** entering the market (summer peak)
- **Serviced Accommodation operators** (high revenue = high tax complexity)
- **Non-UK resident landlords** with UK property income
- **Accountants specialising in property tax** (identified in prospect-list.md)

---

## 2. Key Value Propositions

### Core Benefits

1. **MTD Compliance** — Generates HMRC-ready XML for Making Tax Digital
2. **Multi-Platform Consolidation** — Aggregates data from Airbnb, Booking.com, Vrbo into one format
3. **Time Savings** — Eliminates spreadsheet reconciliation (estimated 5+ hours/quarter per client)
4. **Accuracy** — Reduces transcription errors that trigger HMRC enquiries
5. **Quarterly Reporting** — Supports the MTD quarterly submission cycle

### Emotional Drivers

- **Fear:** HMRC penalties, investigation, fines for incorrect reporting
- **Frustration:** Spreadsheet nightmare, manually reformatting CSV exports
- **Confidence:** Knowing the XML file is correct before submission
- **Time:** Reclaiming hours spent on data entry

### Differentiators vs. Competitors

- Purpose-built for short-term rental (vs. general accounting software)
- Handles multi-platform data (vs. single-platform tools)
- Generates actual HMRC XML (vs. reports that still need manual submission)
