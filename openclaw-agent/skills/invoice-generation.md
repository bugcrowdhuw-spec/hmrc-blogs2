# Skill: Invoice Generation

_Capability to generate professional invoices for rental income, management fees, and other property-related transactions._

## Triggers

- "generate invoice", "create invoice", "invoice me"
- "invoice for", "payment invoice", "receipt"
- "rental invoice", "management fee invoice"

## Invoice Types

### 1. Rental Income Invoice
Generated from recorded rental income entries. Includes:
- Property details
- Rental period
- Income breakdown by date
- Total amount

### 2. Management Fee Invoice
For property management services:
- Management fee calculations
- Additional services
- Total fees due

### 3. Expense Reimbursement Invoice
For expense claims:
- Itemized expenses
- VAT breakdown (if applicable)
- Total reclaimable amount

## Invoice Fields

| Field | Description | Required |
|-------|-------------|----------|
| invoice_number | Unique identifier | Auto-generated |
| date | Invoice date | Yes |
| property_id | Associated property | Yes |
| invoice_type | Type of invoice | Yes |
| line_items | Itemized charges | Yes |
| total | Total amount | Calculated |
| vat | VAT amount | If applicable |

```
**Date:** 8 April 2026
**Type:** Rental Income

- **Name:** London Flat

| Date | Description | Amount |

*Generated by TaxPrep - UK Property Tax Assistant*
```

## CLI Commands

```bash
# Generate rental invoice
taxprep invoice --property-id abc123 --type rental --output invoice.md

# Generate management fee invoice
taxprep invoice --property-id abc123 --type management --date 2026-03-31

# Export to CSV
taxprep invoice --export-csv --output invoices.csv
```

## Invoice Numbering

Format: `INV-YYYYMMDD-XXX`
- YYYYMMDD: Date
- XXX: Sequential number (001-999)

## Export Options

- **Markdown** - Default format
- **PDF** - Requires additional dependencies
- **CSV** - For accounting software import
- **JSON** - For API integration

## Integration

Invoices are automatically:
- Stored in database
- Linked to property records
- Available for report generation
- Exportable for accountant
