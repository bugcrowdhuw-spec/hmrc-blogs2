Property managers face unique HMRC challenges:

Traditional accounting software wasn't built for this complexity. You need tools designed specifically for property management.

## HMRC Compliance Requirements

### Making Tax Digital (MTD)

Since April 2019, VAT-registered businesses must use MTD-compatible software. Property managers must comply if:

- **Turnover exceeds £85,000** — VAT registration threshold
- **You choose to register voluntarily** — Even below threshold

MTD Requirements:

- **Digital record-keeping** — All transactions in approved software
- **Quarterly submissions** — File returns every quarter
- **API submission** — Direct to HMRC's systems

### Digital Record-Keeping

Even below VAT threshold, proper digital records protect you:

- **Transaction-level tracking** — Every income and expense captured
- **Category coding** — Proper tax-deductible classification
- **Receipt attachment** — Digital storage of supporting documents
- **Audit trail** — Complete history of changes

### Report Generation

HMRC requires specific formats:

- **SA Property** — Self Assessment supplementary pages
- **XML reports** — Machine-readable for electronic filing
- **Summary documents** — Year-end totals forSelf Assessment
- **Detailed breakdowns** — Per-property profit calculations

## FAQ

### What happens if I miss an MTD quarterly deadline?

HMRC charges penalties for late filing. First offence: £200. Subsequent late filings within 12 months increase penalties. Persistent late filing triggers higher penalties and potential inspections.

### Can I use spreadsheet software for MTD?

No. Spreadsheets don't meet MTD requirements. HMRC-approved software with API connectivity is mandatory for VAT-registered businesses.

### Do I need MTD if I'm below the VAT threshold?

MTD for income tax is not yet mandatory below £85,000 turnover. However, we strongly recommend MTD-compatible software for all property managers. It's coming for everyone eventually.

### How do I verify software is MTD-approved?

Check HMRC's official software products list. Only approved software can be used for MTD compliance. Search for "HMRC software products list" on gov.uk.
