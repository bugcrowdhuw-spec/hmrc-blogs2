## FAQ

### Can I use the £1,000 Property Income Allowance?

Yes. If your Airbnb income is under £1,000 per year, you can claim the allowance. However, you must still register on HMRC's website if you use this allowance — it's not automatic.

### Do I need to register as self-employed for Airbnb income?

Yes. File a Self Assessment tax return if you earn over £1,000 from Airbnb. Register as soon as you start hosting to avoid penalties.

### What happens if I use multiple platforms?

Your software should combine income from all platforms. Report total income on your Self Assessment, regardless of which platform it came from.

### Can I track expenses separately per property?

Yes. Most UK tax software supports multi-property tracking. Each property generates its own profit calculation for tax purposes.
