# Do You Need to Register for Self Assessment for Airbnb Income?

**If you're earning Airbnb income, you likely need to register.**

## The Rule

Any rental income requires Self Assessment registration - even below Personal Allowance.

## When to Register

Register by 5th October in the tax year you first earn income.

## How to Register

- Go to GOV.UK
- Search "register for Self Assessment"
- Follow steps
- Get activation code by post

## After Registering

- File return by 31st January each year
- Pay tax due by deadline
